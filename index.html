SmartScanCargo

Clique em dois pontos para equalizar uma região livremente ou use a detecção automática para visualizar possíveis regiões suspeitas.

Imagem: nenhuma imagem carregada

Pontos selecionados: 0/2

Último bounding box: nenhum

Controles

  • Escolher imagem no seu dispositivo: abre uma imagem armazenada no computador, celular ou tablet.
  • Imagem exemplo do GitHub: carrega uma das imagens disponíveis na pasta ImagensTest do projeto.
  • Clique esquerdo 1: define o ponto p1.
  • Clique esquerdo 2: define o ponto p2 e equaliza o bounding box.
  • Clique direito: restaura a imagem original.
  • Possible Suspected Region: executa o algoritmo atual e a análise FFT, mostrando os bounding boxes vermelhos sobrepostos dentro da região R.
  • Bounding boxes vermelhos: são apenas indicações; a equalização continua sendo definida por dois cliques, dentro ou fora deles.
  • ESC: baixa a imagem equalizada atual.
